### Locally host

If you want to use `mdbook serve` to locally host the ebook, make sure you have (mdbook have not integrated mermaid in the current version):

```bash
mdbook -V
mdbook v0.4.51
```

and

```bash
mdbook-mermaid -V
mdbook-mermaid 0.15.0
```
